# ХЕХЕ 😂

Магазин мемных футболок. React + Vite + Framer Motion. Без бэкенда — всё живёт в браузере.

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
```

## Что внутри

- **Конструктор** — свой текст и фото на футболке, перёд и спина отдельно, перетаскивание мышью и пальцем, шрифты, начертание, обводка, поворот, отмена (Ctrl+Z).
- **Двуязычие RU / ҚАЗ** — переключается всё, включая надписи на самих футболках.
- **Тёмная тема** — переключатель в шапке, при первом заходе берётся из настроек системы.
- **Оплата картой или кошельком** — без регистрации. С кошелька возвращается 7% коинами.
- **Кошелёк** — баланс, пополнение, история заказов, избранное. Всё в `localStorage`.
- **Два вида карточек** — рисованные в SVG (принт подстраивается под язык и ширину) и фотографии из `public/`.
- Таймер дропа, бегущая строка, «Мне повезёт 🎲», эмодзи-хвост за курсором.

## Оплата пока фейковая

Карта — демонстрационная, никаких полей для ввода номера нет и деньги не списываются:
кнопка имитирует авторизацию и создаёт заказ. Кошелёк проверяет баланс по-настоящему,
но деньги в нём тоже ненастоящие.

Чтобы принимать реальные платежи, нужен бэкенд: сумму нельзя считать на клиенте,
иначе заказ можно оформить на любую цену через консоль браузера.

## Файлы

| Файл | Что там |
|---|---|
| `src/data.js` | каталог, цены, шаблоны, оба перевода — правится здесь |
| `src/App.jsx` | секции, корзина, оплата, кошелёк, тема, состояние |
| `src/components/Designer.jsx` | конструктор: вкладки, история отмены, загрузка фото |
| `src/components/Tshirt.jsx` | футболка в SVG: силуэт, принт, перенос строк, перетаскивание |
| `src/components/Account.jsx` | кошелёк, заказы, избранное |
| `src/components/Logo.jsx` | логотип-смайл, «ржёт» при наведении |
| `src/styles.css` | тема и переменные светлой/тёмной палитры |

## Темы

Палитра задана переменными в `:root`, тёмная — в `:root[data-theme='dark']`.
Ключевые: `--paper` (фон), `--ink` (текст и рамки), `--surface` (карточки),
`--shadow-c` (жёсткие тени), `--band-bg` (тёмные полосы), `--pill-bg` / `--pill-fg`
(активные чипы). Новый цвет добавляй сюда, а не в конкретное правило.

## Как добавить футболку

Один объект в `PRODUCTS` (`src/data.js`): `price`, `rarity`, цвета `color`/`ink`
и блоки `ru` / `kk` с `title`, `print`, `sub`.

Дальше два варианта:

- **Рисованная** — задай `art` (ключ мини-иллюстрации из `Tshirt.jsx`) и `print`.
  `print` разбивается по `\n`, кегль подбирается автоматически.
- **Фотография** — положи файл в `public/` и укажи `photo: '/имя-файла.jpg'`.
  Тогда `art` и `print` не нужны, показывается сама картинка.
